Linux Compression and Archiving Tools

  • tar: Archive files and directories.
  • gzip: Compress files using the GNU zip algorithm.
  • gunzip: Decompress files compressed with gzip.
  • bzip2: Compress files using the Burrows-Wheeler block sorting algorithm.
  • bunzip2: Decompress files compressed with bzip2.
  • zip: Package and compress files.
  • unzip: Extract compressed files in ZIP format.
  • xz: Compress files using the LZMA compression algorithm.
  • unxz: Decompress files compressed with xz.
