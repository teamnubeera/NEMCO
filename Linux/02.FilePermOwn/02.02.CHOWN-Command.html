Linux chown Command Guide

1. Understanding chown

The chown command in Linux is used to change the ownership of files and directories.

2. Basic Syntax

chown [OPTIONS] USER:GROUP FILE

3. Viewing File Ownership

To check file ownership, use:

ls -l filename

4. Changing Ownership

Change Owner Only

chown username filename

Change Owner and Group

chown username:groupname filename

5. Changing Ownership Recursively

Apply ownership change to all files in a directory:

chown -R username:groupname directory/

6. Using chown with Symbolic References

chown --reference=existingfile newfile

7. Common chown Commands

CommandDescription
chown user fileChange file owner
chown user:group fileChange owner and group
chown -R user:group folderRecursively change ownership
