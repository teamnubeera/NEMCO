Linux umask Command Guide

1. Understanding umask

The umask command in Linux is used to set default file permissions for newly created files and directories.

2. Basic Syntax

umask [PERMISSION]

3. Viewing Current umask Value

To check the current umask value, use:

umask

4. Setting umask Value

To change the default file permission mask:

umask 022

5. Understanding umask Values

umask ValueFile Default PermissionsDirectory Default Permissions
000666 (-rw-rw-rw-)777 (rwxrwxrwx)
022644 (-rw-r--r--)755 (rwxr-xr-x)
077600 (-rw-------)700 (rwx------)

6. Applying umask Permanently

To make umask changes permanent, add the command to:

~/.bashrc or ~/.profile

7. Common umask Commands

CommandDescription
umask 022Default for most systems, allows read access for others
umask 077Restricts access to owner only
