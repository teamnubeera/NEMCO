Linux chmod Command Guide

1. File Permissions in Linux

Every file and directory has three types of permissions for three types of users:

User TypeSymbolDescription
OwneruThe user who owns the file
GroupgThe group that owns the file
OthersoAll other users

2. Viewing File Permissions

To check file permissions, use:

ls -l filename

Example output:

-rwxr-xr-- 1 user group 1234 Jan 17 10:00 myfile.sh

3. Changing File Permissions Using chmod

Numeric Mode (Octal Representation)

chmod 754 myfile.sh

Symbolic Mode

chmod u+rwx,go-rwx myfile.sh

4. Changing Permissions Recursively

chmod -R 755 myfolder/

5. Special Permissions

SetUID

chmod u+s myfile

Sticky Bit

chmod +t /tmp

6. Common chmod Commands

CommandDescription
chmod 777 fileEveryone has full access
chmod 755 fileOwner: rwx, Others: r-x
chmod 644 fileOwner: rw-, Others: r--
chmod 700 fileOnly owner has full access

7. Difference Between chmod and chown

chmod changes permissions, while chown changes ownership.

chown user:group myfile
