Linux chgrp Command Guide

1. Understanding chgrp

The chgrp command in Linux is used to change the group ownership of files and directories.

2. Basic Syntax

chgrp [OPTIONS] GROUP FILE

3. Viewing File Group Ownership

To check file group ownership, use:

ls -l filename

4. Changing Group Ownership

chgrp groupname filename

5. Changing Group Ownership Recursively

Apply group ownership change to all files in a directory:

chgrp -R groupname directory/

6. Using chgrp with Symbolic References

chgrp --reference=existingfile newfile

7. Common chgrp Commands

CommandDescription
chgrp group fileChange group ownership
chgrp -R group folderRecursively change group ownership
chgrp --reference=ref_file target_fileChange group to match reference file
