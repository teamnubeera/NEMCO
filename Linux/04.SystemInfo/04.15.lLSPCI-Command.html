lspci Command Overview

lspci Command

  • Lists all PCI devices on the system, including graphics cards, network adapters, and other peripherals.
  • Provides information about the device, such as vendor and device IDs, and device class.

Hands-on Activity:

Run the lspci command in your terminal to list all PCI devices on your system.

Common Options

  • -v: Show verbose output (more detailed information).
  • -nn: Show numeric IDs (device/vendor IDs) instead of names.
  • -k: Show kernel driver information for each device.
  • -t: Display devices in a tree format.

Hands-on Activity:

Run lspci -v to view detailed information about each PCI device.
