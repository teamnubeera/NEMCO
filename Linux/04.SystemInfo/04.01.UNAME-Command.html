Linux uname Command

The uname command in Linux is used to display basic system information, including the kernel name, machine type, and more.

Basic Syntax:

uname [OPTIONS]

Options:

  • -a: Prints all available system information.
  • -s: Prints the kernel name.
  • -n: Prints the network hostname.
  • -r: Prints the kernel release.
  • -v: Prints the kernel version.
  • -m: Prints the machine hardware name.
  • -p: Prints the processor type (if available).
  • -i: Prints the hardware platform (if available).
  • -o: Prints the operating system.

Examples:

1. Print Kernel Name:

$ uname -s
Linux

Explanation: Displays the kernel name of the system.

2. Print Kernel Release:

$ uname -r
5.15.0-56-generic

Explanation: Displays the version of the Linux kernel currently running.

3. Print All System Information:

$ uname -a
Linux ubuntu 5.15.0-56-generic #62-Ubuntu SMP Fri Nov 25 18:24:45 UTC 2022 x86_64 x86_64 x86_64 GNU/Linux

Explanation: Displays all available system information, including kernel name, hostname, kernel release, version, machine type, and more.

4. Print Machine Hardware Name:

$ uname -m
x86_64

Explanation: Displays the hardware architecture of the system.

5. Print Operating System:

$ uname -o
GNU/Linux

Explanation: Displays the operating system running on the system.

Practical Uses:

  • System Identification: Quickly check the kernel, machine, or OS details.
  • Debugging: Determine the system information while troubleshooting issues.
  • Scripting: Include system details in automated scripts for dynamic behavior.

Script Example:

#!/bin/bash
# Check the kernel version and machine architecture
KERNEL=$(uname -r)
MACHINE=$(uname -m)
echo "Kernel Version: $KERNEL"
echo "Machine Architecture: $MACHINE"

Output:

Kernel Version: 5.15.0-56-generic
Machine Architecture: x86_64

Summary:

  • uname: Displays system information.
  • -a: Prints all available details.
  • -s: Prints the kernel name.
  • -r: Prints the kernel release.
  • -m: Prints the machine hardware name.
  • -o: Prints the operating system.

Tip: Combine uname with other commands to gather comprehensive system details.
