lsblk Command Overview

lsblk Command

  • Lists information about all available block devices (e.g., hard drives, SSDs, partitions).
  • Displays details like device name, size, type, and mount point.

Hands-on Activity:

Run the lsblk command in your terminal to list all block devices and their details.

Common Options

  • -f: Display file system information.
  • -o: Specify columns to display (e.g., lsblk -o NAME,SIZE,TYPE).
  • -l: Display devices in a list format.
  • -a: Show all devices, including empty ones.

Hands-on Activity:

Run lsblk -f to display block devices with file system information.
