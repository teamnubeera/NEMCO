lsusb Command Overview

lsusb Command

  • Lists all USB devices connected to the system.
  • Displays details like the USB device ID, device name, and manufacturer.

Hands-on Activity:

Run the lsusb command in your terminal to list all USB devices currently connected to your system.

Common Options

  • -v: Show verbose output (detailed information about each USB device).
  • -s : Show information for a specific device (e.g., lsusb -s 002:003).
  • -d : Show information for devices with a specific vendor and device ID.
  • -t: Display USB devices in a tree format.

Hands-on Activity:

Run lsusb -v to view detailed information about each connected USB device.
