free Command Overview

free Command

  • Displays memory usage on a Linux/Unix system.
  • Shows total, used, free, shared, buffers, and cached memory.

Hands-on Activity:

Run the free -h command in your terminal to view memory usage in human-readable format.

Common Options

  • -h: Human-readable format (e.g., MB, GB).
  • -m: Display memory in MB.
  • -g: Display memory in GB.
  • -s : Continuously display memory status at specified intervals.

Hands-on Activity:

Run free -m to view memory usage in MB and free -s 2 to update memory statistics every 2 seconds.
