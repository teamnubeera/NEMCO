Linux Disk Management Tools

  • fdisk: Partition table manipulator (for MBR partitioning).
  • parted: A more advanced partitioning tool.
  • mkfs: Create a file system on a device.
  • mount: Mount a file system.
  • umount: Unmount a file system.
  • fsck: File system consistency check.
  • blkid: Locate/print block device attributes.
  • tune2fs: Adjust tunable parameters of ext2/ext3/ext4 file systems.
