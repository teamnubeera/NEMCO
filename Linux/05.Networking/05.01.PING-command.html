Linux ping Command Guide

1. Understanding ping

The ping command in Linux is used to send ICMP echo request packets to a network host to check connectivity and measure response time.

2. Basic Syntax

ping [OPTIONS] DESTINATION

3. Checking Connectivity

To check if a host is reachable:

ping example.com

4. Specifying Number of Packets

To limit the number of packets sent:

ping -c 4 example.com

5. Setting Packet Size

ping -s 64 example.com

6. Common ping Commands

CommandDescription
ping hostnameSend continuous ping requests
ping -c 4 hostnameSend 4 ping requests
ping -i 2 hostnameSend ping requests every 2 seconds
ping -s 128 hostnameSet packet size to 128 bytes
