Traceroute Command: Trace the Route Packets Take to a Network Host

The traceroute command in Linux is used to trace the path packets take to reach a network destination, displaying each hop along the route.

Basic Usage

  • traceroute example.com - Trace the route to example.com.
  • traceroute -4 example.com - Use IPv4 only.
  • traceroute -6 example.com - Use IPv6 only.

Advanced Options

  • traceroute -m 20 example.com - Set the maximum number of hops to 20.
  • traceroute -q 3 example.com - Set the number of queries per hop to 3.
  • traceroute -n example.com - Display numeric IP addresses instead of resolving hostnames.

Alternative Commands

  • tracepath example.com - A simpler alternative to traceroute.
  • mtr example.com - Combines traceroute and ping for real-time monitoring.
