NSLookup Command: Query DNS Servers for Information

The nslookup command in Linux and Windows is used to query Domain Name System (DNS) servers to obtain domain name or IP address mapping and other DNS records.

Basic Usage

  • nslookup example.com - Get the IP address of example.com.
  • nslookup 8.8.8.8 - Reverse lookup: Find the domain name associated with an IP.

Advanced Options

  • nslookup -type=A example.com - Query the A record (IPv4 address).
  • nslookup -type=AAAA example.com - Query the AAAA record (IPv6 address).
  • nslookup -type=MX example.com - Retrieve mail exchange (MX) records.
  • nslookup -type=NS example.com - Get the name servers for a domain.

Using a Specific DNS Server

  • nslookup example.com 8.8.8.8 - Query example.com using Google's DNS server.
  • nslookup example.com 1.1.1.1 - Query using Cloudflareâ€™s DNS server.

Interactive Mode

  • Run nslookup without arguments to enter interactive mode.
  • Type set type=MX then example.com to query MX records.
  • Type exit to leave interactive mode.
