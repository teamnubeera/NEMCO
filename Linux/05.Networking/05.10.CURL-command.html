Curl Command: Transfer Data from or to a Server

The curl command in Linux is used to transfer data from or to a server, supporting multiple protocols like HTTP, HTTPS, FTP, SCP, and SFTP.

Basic Usage

  • curl http://example.com - Fetch the content of a web page.
  • curl -o file.zip http://example.com/file.zip - Download a file and save it with a specified name.

Handling Different Protocols

  • curl ftp://example.com/file.zip - Download a file using FTP.
  • curl -u user:password ftp://example.com/file.zip - Authenticate and download from FTP.

Sending Data

  • curl -d "param1=value1¶m2=value2" -X POST http://example.com/api - Send POST request with data.
  • curl -H "Content-Type: application/json" -d '{"key":"value"}' -X POST http://example.com/api - Send JSON data via POST.

Downloading Multiple Files

  • curl -O http://example.com/file1.zip -O http://example.com/file2.zip - Download multiple files.

Resuming Downloads

  • curl -C - -O http://example.com/file.zip - Resume an interrupted download.

Fetching Headers

  • curl -I http://example.com - Retrieve HTTP headers only.
