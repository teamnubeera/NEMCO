Netstat Command: Print Network Connections, Routing Tables, etc.

The netstat command in Linux is used to display network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.

Basic Usage

  • netstat -a - Show all active connections.
  • netstat -t - Display TCP connections.
  • netstat -u - Display UDP connections.
  • netstat -r - Show the kernel routing table.

Displaying Network Statistics

  • netstat -i - Show network interfaces and statistics.
  • netstat -s - Display summary statistics for each protocol.

Monitoring Active Connections

  • netstat -an - Show all active connections with numeric addresses.
  • netstat -tn - Show active TCP connections numerically.
