SS Command: Display Socket Statistics (Replacement for Netstat)

The ss command in Linux is used to display detailed socket statistics and is considered a faster alternative to netstat.

Basic Usage

  • ss -a - Show all sockets.
  • ss -t - Display TCP sockets.
  • ss -u - Display UDP sockets.
  • ss -l - Show listening sockets.

Displaying Connection Details

  • ss -s - Display summary statistics.
  • ss -tn - Show TCP sockets numerically.
  • ss -p - Show process using the sockets.

Filtering Connections

  • ss -t state established - Show established TCP connections.
  • ss -u state connected - Show active UDP connections.
