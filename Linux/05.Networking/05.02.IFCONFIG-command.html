Linux ifconfig Command Guide

1. Understanding ifconfig

The ifconfig command in Linux is used to display and configure network interfaces. It is part of the net-tools package and is mostly replaced by ip command in modern distributions.

2. Basic Syntax

ifconfig [INTERFACE] [OPTIONS]

3. Viewing Network Interfaces

To display all active network interfaces:

ifconfig

4. Enabling or Disabling an Interface

To enable a network interface:

ifconfig eth0 up

To disable a network interface:

ifconfig eth0 down

5. Assigning an IP Address

ifconfig eth0 192.168.1.100 netmask 255.255.255.0

6. Common ifconfig Commands

CommandDescription
ifconfigDisplay active network interfaces
ifconfig eth0 upEnable network interface eth0
ifconfig eth0 downDisable network interface eth0
ifconfig eth0 192.168.1.100Assign IP address to eth0
ifconfig eth0 mtu 1500Set MTU size for eth0
