DIG Command: Query DNS Information in More Detail

The dig (Domain Information Groper) command in Linux is used to query DNS servers for detailed domain and IP address information.

Basic Usage

  • dig example.com - Get detailed DNS information about example.com.
  • dig +short example.com - Display only the IP address.
  • dig -x 8.8.8.8 - Perform a reverse lookup on an IP address.

Querying Specific DNS Records

  • dig example.com A - Query the A record (IPv4 address).
  • dig example.com AAAA - Query the AAAA record (IPv6 address).
  • dig example.com MX - Retrieve mail exchange (MX) records.
  • dig example.com NS - Get the name servers for a domain.

Using a Specific DNS Server

  • dig @8.8.8.8 example.com - Query example.com using Google's DNS server.
  • dig @1.1.1.1 example.com - Query using Cloudflareâ€™s DNS server.

Advanced Options

  • dig +trace example.com - Trace the DNS resolution path.
  • dig +noall +answer example.com - Show only the answer section.
  • dig example.com ANY - Get all available DNS records for a domain.
