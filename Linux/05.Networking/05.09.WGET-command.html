Wget Command: Non-Interactive Download Utility (HTTP, FTP, etc.)

The wget command in Linux is used to download files from the web in a non-interactive way, supporting HTTP, HTTPS, and FTP protocols.

Basic Usage

  • wget http://example.com/file.zip - Download a file from a URL.
  • wget -O newname.zip http://example.com/file.zip - Save the file with a different name.

Downloading Multiple Files

  • wget -i urls.txt - Download multiple URLs listed in a file.

Resuming Downloads

  • wget -c http://example.com/file.zip - Resume an interrupted download.

Downloading in the Background

  • wget -b http://example.com/file.zip - Run download in the background.

Limiting Download Speed

  • wget --limit-rate=200k http://example.com/file.zip - Limit speed to 200KB/s.

Mirroring a Website

  • wget --mirror -p --convert-links -P ./localdir http://example.com/ - Mirror a website locally.
