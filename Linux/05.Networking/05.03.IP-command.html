IP Command: Show/Manipulate Routing, Devices, and Tunnels

The ip command in Linux is used to show and manipulate routing, network devices, and tunnels.

Basic Usage

  • ip addr show - Display IP addresses assigned to all interfaces.
  • ip link show - Show details of network interfaces.
  • ip route show - Display the routing table.

Manipulating Network Interfaces

  • ip link set eth0 up - Enable the eth0 interface.
  • ip link set eth0 down - Disable the eth0 interface.
  • ip addr add 192.168.1.100/24 dev eth0 - Assign an IP address to eth0.

Routing

  • ip route add default via 192.168.1.1 - Set default gateway.
  • ip route del 192.168.1.0/24 - Remove a specific route.
