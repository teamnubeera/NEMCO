ip addr: Show IP Address Information of Interfaces

The ip addr command is used to display and manage IP address information for network interfaces in Linux-based operating systems. It is part of the iproute2 package, which provides tools for network management.

Basic ip addr Syntax

The basic syntax for the ip addr command is:

ip addr [options]

By default, running ip addr without options will show the IP addresses of all active network interfaces on the system.

Examples

Show All IP Addresses

To display the IP addresses of all network interfaces on your system, simply run:

ip addr

This command will display information about each network interface, including IPv4 and IPv6 addresses, along with their status (UP or DOWN), network masks, and other details.

Show Specific Interface Information

If you want to display the IP address information for a specific interface (e.g., eth0 or wlan0), you can specify the interface name:

ip addr show dev eth0

For example, to view information for the eth0 interface:

ip addr show dev eth0

Show IPv4 or IPv6 Addresses Only

If you want to see only IPv4 or IPv6 addresses, you can use the inet (for IPv4) or inet6 (for IPv6) filters:

ip addr show | grep inet

This will show only IPv4 addresses. Similarly, to show IPv6 addresses:

ip addr show | grep inet6

Show Interface Status

If you're interested in seeing whether interfaces are up or down, use the following command:

ip addr show

This will include the status (e.g., UP, DOWN) of the interfaces alongside their IP addresses.

Show Additional Information

If you want to display more detailed information about each interface, including MAC addresses and interface statistics, run:

ip addr show

This will show more information about each interface, including the MAC address (link/ether) and the broadcast address.

Useful ip addr Options

  • show: Display the address information for the specified interface.
  • dev [interface]: Specify the network interface (e.g., eth0, wlan0) whose IP information you want to display.
  • inet: Filter and show IPv4 addresses only.
  • inet6: Filter and show IPv6 addresses only.
  • up: Show only interfaces that are currently active (i.e., UP).
  • down: Show only interfaces that are currently inactive (i.e., DOWN).

Security Considerations

The ip addr command only displays information about network interfaces. However, be mindful that on some systems, network configurations might include sensitive information, such as IP addresses or routing details. Ensure that only authorized users have access to this information, especially in production environments.

Note: If you're working with network configurations that require elevated privileges, you may need to run sudo ip addr to view or modify network interface settings.

Conclusion

The ip addr command is a powerful and essential tool for managing network interfaces and viewing IP address configurations on Linux-based systems. Whether you're troubleshooting connectivity issues or configuring network interfaces, this command provides you with all the necessary details about the system's network setup.

For more advanced usage and options, refer to the official ip command manual page.
