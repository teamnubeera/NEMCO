stat - Display File or File System Status

The stat command provides detailed information about a file or directory, including permissions, size, and timestamps.

Purpose:

To display comprehensive metadata about a file or file system.

Usage:

stat file.txt

This command displays detailed information about the file named file.txt.

Example Output:

  File: file.txt
  Size: 1234       Blocks: 8          IO Block: 4096   regular file
Device: 802h/2050d   Inode: 1234567     Links: 1
Access: 2025-01-17 12:34:56.000000000 +0000
Modify: 2025-01-15 09:15:42.000000000 +0000
Change: 2025-01-15 09:15:42.000000000 +0000

Key Fields in the Output:

  • File: The name of the file or directory.
  • Size: The size of the file in bytes.
  • Blocks: The number of blocks allocated to the file.
  • IO Block: The size of each block for I/O operations.
  • Device: The device number where the file resides.
  • Inode: The inode number of the file.
  • Links: The number of hard links to the file.
  • Access: The last access timestamp.
  • Modify: The last modification timestamp.
  • Change: The last status change timestamp.

Notes:

The stat command is particularly useful for troubleshooting file issues and understanding detailed file metadata.
