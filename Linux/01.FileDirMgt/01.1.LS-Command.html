ls Command in Detail with Multiple Examples

Basic Syntax:

ls [OPTION] [FILE/DIRECTORY...]

Common Options with Examples

1. ls (basic usage)

Lists files and directories in the current directory.

ls

Output:

file1.txt  file2.txt  dir1  dir2

2. -l (long listing format)

Displays detailed information about files such as permissions, owner, group, size, and modification time.

ls -l

Output:

-rw-r--r-- 1 user user  123 Jan 16 10:00 file1.txt
drwxr-xr-x 2 user user 4096 Jan 16 09:45 dir1

3. -a (show all files, including hidden files)

Displays all files, including hidden files (those starting with .).

ls -a

Output:

.  ..  .hidden_file  file1.txt  dir1

4. -la or -al (long listing with hidden files)

Combines -l and -a to show all files, including hidden ones, with detailed information.

ls -la

Output:

drwxr-xr-x  2 user user 4096 Jan 16 09:45 .
drwxr-xr-x 10 user user 4096 Jan 16 09:00 ..
-rw-r--r--  1 user user  123 Jan 16 10:00 .hidden_file
-rw-r--r--  1 user user  123 Jan 16 10:00 file1.txt
drwxr-xr-x  2 user user 4096 Jan 16 09:45 dir1

5. -h (human-readable sizes)

Displays file sizes in a more human-readable format (e.g., KB, MB, GB) when used with -l.

ls -lh

Output:

-rw-r--r-- 1 user user 1.2K Jan 16 10:00 file1.txt
drwxr-xr-x 2 user user 4.0K Jan 16 09:45 dir1

6. -R (recursive listing)

Lists files and directories recursively, showing the contents of all subdirectories.

ls -R

Output:

dir1:
file1.txt

dir2:
file2.txt

7. -S (sort by file size)

Sorts files by size in descending order.

ls -S

Output:

largefile.txt  mediumfile.txt  smallfile.txt

8. -t (sort by modification time)

Sorts files by modification time, with the most recently modified files appearing first.

ls -t

Output:

file2.txt  file1.txt  oldfile.txt

9. -r (reverse order)

Reverses the sorting order. Can be used with other sorting options like -t or -S.

ls -r

Output:

dir2  dir1  file2.txt  file1.txt

10. -1 (one entry per line)

Displays one file or directory per line.

ls -1

Output:

file1.txt
file2.txt
dir1
dir2

11. -d (list directories themselves, not their contents)

When listing directories, shows the directory itself rather than its contents.

ls -d */

Output:

dir1/  dir2/

12. --color (enable colorized output)

Colorizes the output to help distinguish between files, directories, and symbolic links.

ls --color

13. -F (append a special character)

Adds a special character to indicate the type of each file. For example, / for directories, * for executables, @ for symlinks.

ls -F

Output:

file1.txt  dir1/  script.sh*  symlink@

14. -i (show inode numbers)

Displays the inode number of each file.

ls -i

Output:

12345678 file1.txt
12345679 dir1

15. --time=access (sort by last access time)

Displays files sorted by last access time instead of modification time.

ls --time=access

Example Combinations:

1. List all files, including hidden, with detailed information and human-readable sizes:

ls -lha

2. List files in a specific directory with detailed information:

ls -l /home/user/

3. List files recursively and sort by modification time:

ls -Rt

4. List files sorted by size, with directories listed last:

ls -Sl

5. List files and directories in reverse alphabetical order:

ls -r

Conclusion:

The ls command is a versatile tool for listing files and directories in Unix/Linux. By combining various options, you can customize the output to fit your needs, whether you're viewing hidden files, sorting by size or time, or displaying detailed file attributes.
