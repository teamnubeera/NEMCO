tree - List Directories and Files in a Tree-like Format

Purpose: Displays files and directories in a hierarchical tree format.

Usage

  • tree - Lists files and directories in the current directory in a tree structure.
  • tree /path/to/directory - Lists files in the specified directory in tree format.

Examples

Example 1: Basic Usage

$ tree
.
├── file1.txt
├── file2.txt
└── folder1
    ├── file3.txt
    └── file4.txt

Example 2: Specifying a Directory

$ tree /home/user/documents
/home/user/documents
├── report.docx
├── data
│   ├── data1.csv
│   └── data2.csv
└── images
    ├── image1.png
    └── image2.png

Notes

  • The tree command may not be installed by default on all systems. Install it using your package manager, e.g., sudo apt install tree on Debian-based systems.
  • Use tree -L N to limit the depth of the tree to N levels.
  • Use tree -d to display only directories.
