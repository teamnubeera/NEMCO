Linux touch Command

The touch command in Linux is primarily used to create empty files. It can also be used to update the timestamp of existing files.

Basic Syntax:

touch [OPTIONS] FILE_NAME

Options:

  • touch: Creates a new empty file or updates the timestamp of an existing file.
  • touch -a: Updates only the access time of the file.
  • touch -m: Updates only the modification time of the file.
  • touch -c: Does not create a file if it does not exist; only updates the timestamp of existing files.
  • touch -t: Sets a specific time for the file in the format [[CC]YY]MMDDhhmm[.ss].

Examples:

1. Create an Empty File:

$ touch newfile.txt

Explanation: Creates an empty file named newfile.txt if it does not already exist.

2. Update Access Time:

$ touch -a existingfile.txt

Explanation: Updates only the access time of existingfile.txt.

3. Update Modification Time:

$ touch -m logfile.txt

Explanation: Updates only the modification time of logfile.txt.

4. Do Not Create a File:

$ touch -c nonexistentfile.txt

Explanation: Does nothing if nonexistentfile.txt does not exist.

5. Set a Specific Timestamp:

$ touch -t 202401150830.00 report.txt

Explanation: Sets the timestamp of report.txt to January 15, 2024, 08:30:00 AM.

Practical Uses:

  • File Creation: Quickly create multiple empty files.
  • Timestamp Management: Update file timestamps for scripting purposes.
  • Testing: Create test files for development or automation scripts.

Script Example:

#!/bin/bash
# Create multiple empty files for a project
for file in index.html style.css script.js; do
    touch $file
    echo "$file created."
done

Output:

index.html created.
style.css created.
script.js created.

Summary:

  • touch: Creates an empty file or updates timestamps.
  • touch -a: Updates the access time of a file.
  • touch -m: Updates the modification time of a file.
  • touch -c: Avoids creating a file if it does not exist.
  • touch -t: Sets a specific timestamp for a file.

Note: Be cautious when using touch with existing files as it can change their timestamps.
