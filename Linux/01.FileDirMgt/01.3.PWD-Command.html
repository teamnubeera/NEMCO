Linux pwd Command

The pwd command in Linux is used to print the current working directory. This is the directory in which your shell or terminal session is currently located.

Basic Syntax:

pwd [OPTION]

Options:

  • pwd: Displays the full path of the current directory.
  • pwd -L: Displays the logical path, including symbolic links.
  • pwd -P: Displays the physical path, resolving symbolic links.

Examples:

1. Basic Usage:

$ pwd
/home/zaid/Documents

Explanation: Displays the absolute path of the current working directory.

2. Using -L Option (Logical Path):

$ pwd -L
/home/zaid/shortcut

Explanation: Displays the logical path, including symbolic links.

3. Using -P Option (Physical Path):

$ pwd -P
/home/zaid/realpath

Explanation: Resolves symbolic links and shows the actual directory path.

4. Symbolic Link Behavior:

# Create a symbolic link
$ ln -s /home/zaid/realpath /home/zaid/shortcut

# Logical path
$ cd /home/zaid/shortcut
$ pwd -L
/home/zaid/shortcut

# Physical path
$ pwd -P
/home/zaid/realpath

Practical Uses:

  • Navigation: Confirm the current directory after using cd commands.
  • Scripting: Dynamically get the current directory for file management or logging.
  • Troubleshooting: Identify the real path of directories, especially when symbolic links are involved.

Script Example:

#!/bin/bash
# Print the current working directory and store it in a variable
current_dir=$(pwd)

echo "Current directory is: $current_dir"

Output:

Current directory is: /home/zaid/scripts

Summary:

  • pwd: Displays the absolute path of the current directory.
  • pwd -L: Displays the logical path, including symbolic links.
  • pwd -P: Displays the physical path, resolving symbolic links.
