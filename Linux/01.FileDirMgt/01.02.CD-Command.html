cd Command in Detail with Multiple Examples

Basic Syntax:

cd [DIRECTORY]

Common Usages with Examples

1. cd (change to home directory)

If cd is used without any argument, it changes the current working directory to the user's home directory.

cd

Example Output: (changes the directory to the home directory)

~/

2. cd [directory] (change to a specific directory)

Changes the current working directory to the specified directory. The directory name can be either relative or absolute.

cd /path/to/directory

Example Output: (changes the directory to the specified path)

/path/to/directory

3. cd .. (go up one directory level)

Moves up one directory level (goes to the parent directory).

cd ..

Example Output: (moves to the parent directory)

/home/user/

4. cd ~ (go to home directory)

The ~ (tilde) symbol represents the current user's home directory. Using cd ~ changes the current directory to the home directory.

cd ~

Example Output: (changes to the home directory)

~/

5. cd - (switch to the previous directory)

Changes the directory to the previous working directory. This is helpful when toggling between two directories.

cd -

Example Output: (changes to the last directory you were in)

/path/to/previous/directory

6. cd / (change to the root directory)

Changes the current working directory to the root directory.

cd /

Example Output: (changes to the root directory)

/

7. cd [relative path] (change to a subdirectory)

Uses a relative path to navigate to a subdirectory from the current working directory. The relative path does not start with a slash (/).

cd foldername

Example Output: (changes to the subdirectory under the current directory)

/home/user/foldername

8. cd ./foldername (use relative path explicitly)

Explicitly uses the current directory by including ./ in the path. This is useful when the directory name starts with a dot (.) or if you want to make the relative path clearer.

cd ./foldername

Example Output: (changes to the folder under the current directory)

/home/user/foldername

Example Combinations:

1. Change to a specific directory and then go to its parent directory:

cd /path/to/directory
cd ..

2. Switch to the previous directory after navigating to another directory:

cd /home/user/documents
cd -

3. Navigate to the root directory and then to the home directory:

cd /
cd ~

Conclusion:

The cd command is one of the most frequently used commands in Unix/Linux systems for navigating between directories. Understanding how to use absolute and relative paths, as well as special symbols like ~ and -, can significantly improve your efficiency when working with the file system. The cd command is simple, yet crucial for directory navigation.
