Linux: Using the awk Command for Text Processing and Pattern Scanning

The awk command is a powerful text processing tool used for pattern scanning and processing. It is used to manipulate text, extract data, perform calculations, and generate reports based on text patterns.

Basic Command Usage

awk 'pattern { action }' filename

The awk command operates by scanning each line of the file filename. When it finds a line that matches the given pattern, it executes the specified action.

Commonly Used Features with awk

  • $0: Represents the entire current line.
  • $1, $2, ...: Represents the first, second, and subsequent fields (columns) of the current line.
  • print: Prints text or values, often used to display results.
  • NR: Represents the current line number (record number).
  • FS: Field separator variable (default is whitespace or tab).
  • OFS: Output field separator (default is a space).

Example Usage

1. Print Specific Columns

awk '{print $1, $3}' file.txt

This command prints the first and third columns from file.txt.

2. Print Line Numbers with Data

awk '{print NR, $0}' file.txt

This command prints the line number followed by the entire line from file.txt.

3. Use a Pattern to Match Lines

awk '/pattern/ {print $0}' file.txt

This command prints all lines from file.txt that contain the specified pattern.

4. Print Columns with a Custom Field Separator

awk 'BEGIN {FS=":"} {print $1, $3}' file.txt

This command sets the field separator to : and prints the first and third columns from file.txt.

5. Perform Calculations

awk '{print $1 + $2}' file.txt

This command adds the values in the first and second columns and prints the result.

6. Using NR for Line Numbers

awk '{print NR, $0}' file.txt

This command prints the line number and the entire content of each line in file.txt.

7. Use awk to Filter and Print Specific Patterns

awk '$2 > 50 {print $1, $2}' file.txt

This command prints the first and second columns where the value in the second column is greater than 50.

Conclusion

The awk command is a versatile tool for text processing and data extraction in Linux. Whether you are extracting columns, performing calculations, or filtering data based on patterns, awk can help you manipulate text effectively. Experiment with these examples to gain deeper control over text data in your files.
