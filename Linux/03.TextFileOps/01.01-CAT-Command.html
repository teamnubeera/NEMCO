Linux: Using the cat Command to Concatenate and Display File Contents

The cat command is used to display the contents of files, concatenate multiple files together, or create new files. It's one of the most commonly used commands in Linux for file viewing and manipulation.

Basic Command Usage

cat filename

This will display the contents of the file filename directly in the terminal.

Commonly Used Options with cat

  • -n: Number all output lines.
  • -b: Number non-blank output lines (skips blank lines).
  • -E: Display "$" at the end of each line to visualize line breaks.
  • -T: Display TAB characters as ^I.

Example Usage

1. Display a Single File

cat index.html

This command displays the contents of the index.html file in the terminal.

2. Display Multiple Files

cat file1.txt file2.txt

This concatenates file1.txt and file2.txt and displays their combined contents in the terminal.

3. Number Lines in Output

cat -n index.html

This command displays the contents of index.html, numbering each line for reference.

4. Show End-of-Line Characters

cat -E index.html

This command will show "$" at the end of each line to visualize the line breaks.

5. Create a New File

cat > newfile.txt

This command will create a new file newfile.txt and allow you to type text directly into it. Press Ctrl + D to save and exit.

Conclusion

The cat command is a versatile tool for working with file contents. It can be used to display, combine, create, and even manipulate files directly in the terminal. Experiment with the various options to gain more control over your file viewing tasks.
