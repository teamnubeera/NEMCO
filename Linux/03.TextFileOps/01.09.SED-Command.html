Linux: Using the sed Command for Filtering and Transforming Text

The sed command is a stream editor used to filter and transform text in Linux. It allows you to perform text manipulation, such as substitutions, deletions, and insertions, on files or input streams.

Basic Command Usage

sed 's/pattern/replacement/' filename

This command replaces the first occurrence of pattern with replacement in the filename. By default, it only replaces the first occurrence on each line.

Commonly Used Options with sed

  • -i: Edit the file in place (without this option, sed outputs the result to the terminal).
  • -n: Suppresses automatic printing of the pattern space (use with p to print specific lines).
  • s: Substitute a pattern with a replacement.
  • d: Delete lines that match a pattern.
  • a: Append text after a line.
  • i: Insert text before a line.

Example Usage

1. Substitute Text

sed 's/old/new/' file.txt

This command replaces the first occurrence of old with new in each line of file.txt.

2. Replace All Occurrences

sed 's/old/new/g' file.txt

The g flag makes the substitution global, replacing all occurrences of old with new in each line.

3. Edit File In Place

sed -i 's/old/new/g' file.txt

This command performs the replacement and updates file.txt directly without needing to redirect the output.

4. Delete Lines Matching a Pattern

sed '/pattern/d' file.txt

This command deletes all lines containing pattern from file.txt.

5. Print Specific Lines

sed -n '1,3p' file.txt

This command prints only lines 1 through 3 from file.txt.

6. Append Text After a Line

sed '/pattern/a\New text' file.txt

This command appends the text New text after every line that contains pattern.

7. Insert Text Before a Line

sed '/pattern/i\New text' file.txt

This command inserts the text New text before every line that contains pattern.

Conclusion

The sed command is a powerful tool for text processing in Linux. With its wide range of functions and options, you can automate and streamline text editing tasks. Practice using these commands to filter and transform text files according to your needs.
