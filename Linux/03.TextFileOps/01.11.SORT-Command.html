Linux: Using the sort Command to Sort Lines in Files

The sort command is used to sort lines of text in files or input streams in Linux. It can arrange data in ascending or descending order and supports sorting based on various criteria such as numeric, lexicographic, and field-based sorting.

Basic Command Usage

sort filename

This command sorts the lines of the file filename in ascending order by default.

Commonly Used Options with sort

  • -n: Sorts lines based on numeric values (useful for sorting numbers).
  • -r: Reverses the sorting order (sorts in descending order).
  • -k: Specifies a key or column to sort by (default is to sort by the entire line).
  • -u: Removes duplicate lines from the output.
  • -t: Specifies a delimiter to use for sorting by columns (default is space or tab).
  • -b: Ignores leading blanks while sorting.
  • -f: Makes the sort case-insensitive.

Example Usage

1. Sort Lines in Ascending Order

sort file.txt

This command sorts the lines in file.txt in ascending lexicographic order (alphabetically for text or numerically for numbers).

2. Sort Lines in Descending Order

sort -r file.txt

This command sorts the lines in file.txt in descending order.

3. Sort Numerically

sort -n file.txt

This command sorts the lines of file.txt in numeric order, which is useful for sorting numbers.

4. Sort by a Specific Column

sort -k 2 file.txt

This command sorts file.txt by the second column of each line.

5. Sort with a Specific Delimiter

sort -t ":" -k 2 file.txt

This command sorts file.txt by the second column, using the colon (:) as a delimiter.

6. Remove Duplicate Lines

sort -u file.txt

This command sorts file.txt and removes any duplicate lines from the output.

7. Case-Insensitive Sorting

sort -f file.txt

This command sorts file.txt without considering case (making it case-insensitive).

8. Sort with Multiple Keys

sort -k 1,1 -k 2,2 file.txt

This command sorts file.txt first by the first column and then by the second column.

Conclusion

The sort command is an essential tool for organizing and arranging data in Linux. Whether you are sorting text alphabetically, numerically, or by specific fields, the sort command offers flexibility and powerful options for managing text data. Practice using these examples to gain more control over sorting data in your files.
