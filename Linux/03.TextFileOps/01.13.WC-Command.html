The `wc` Command

The `wc` (word count) command in Linux is used to display the number of lines, words, and bytes in a file. The syntax for using the `wc` command is as follows:

        wc [OPTION] [FILE...]

Options:

  • -l: Counts the number of lines in the file.
  • -w: Counts the number of words in the file.
  • -c: Counts the number of bytes in the file.
  • -m: Counts the number of characters in the file.
  • -L: Shows the length of the longest line.

Examples:

Counting the lines in a file:

        wc -l filename.txt

Counting the words in a file:

        wc -w filename.txt

Counting the bytes in a file:

        wc -c filename.txt

Output:

The `wc` command will display the count of lines, words, and bytes. For example:

        3  15  120 filename.txt

This output means that the file contains 3 lines, 15 words, and 120 bytes.
