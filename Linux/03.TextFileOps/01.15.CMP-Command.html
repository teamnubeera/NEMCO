Compare Two Files with the cmp Command

The cmp command in Linux is used to compare two files byte by byte. It is useful for detecting the first difference between two files. If the files are identical, no output is displayed. If they differ, it shows the location of the first differing byte.

Syntax:

cmp [options] file1 file2

Example:

cmp file1.txt file2.txt

If the files are identical, there is no output. If they are different, the command will output something like this:

file1.txt file2.txt differ: byte 5, line 1

The output shows the byte number and the line where the difference is found. You can use various options to control the behavior of cmp, such as:

  • -b: Print byte numbers in decimal.
  • -l: Print all differing bytes.
  • -s: Suppress output, only return the exit status.

How to Use cmp:

  1. Open a terminal in your Linux system.
  2. Navigate to the directory where the files are located using cd.
  3. Run the cmp command with the file names as arguments.
  4. Review the output to see if the files are identical or where they differ.
