Linux: Using the uniq Command to Report or Omit Repeated Lines

The uniq command in Linux is used to filter out or report repeated lines from a sorted input. It is particularly useful for eliminating duplicate lines in text files or input streams.

Basic Command Usage

uniq [options] filename

This command removes or reports repeated lines from the file filename. By default, it only removes adjacent duplicate lines, so input must be sorted first for proper functionality.

Commonly Used Options with uniq

  • -c: Prefixes lines with the number of occurrences of each line.
  • -d: Prints only lines that are repeated.
  • -u: Prints only lines that are unique (not repeated).
  • -i: Ignores case when comparing lines.
  • -f NUM: Skips the first NUM fields when comparing lines.
  • -s NUM: Skips the first NUM characters when comparing lines.
  • -w NUM: Compares up to NUM characters per line.

Example Usage

1. Remove Duplicate Lines

uniq file.txt

This command removes consecutive duplicate lines from file.txt and prints only unique lines.

2. Show Number of Occurrences of Each Line

uniq -c file.txt

This command prefixes each line with the number of occurrences of that line in file.txt.

3. Print Only Duplicate Lines

uniq -d file.txt

This command prints only the lines that are repeated in file.txt.

4. Print Only Unique Lines

uniq -u file.txt

This command prints only the lines that are unique in file.txt, omitting any duplicates.

5. Ignore Case When Comparing Lines

uniq -i file.txt

This command treats lines as identical, even if they differ only in case (uppercase vs. lowercase).

6. Skip Fields When Comparing Lines

uniq -f 2 file.txt

This command skips the first two fields (columns) when comparing lines in file.txt.

7. Skip Characters When Comparing Lines

uniq -s 3 file.txt

This command skips the first three characters when comparing lines in file.txt.

8. Compare Only a Specific Number of Characters

uniq -w 5 file.txt

This command compares only the first 5 characters of each line in file.txt.

Conclusion

The uniq command is a useful tool for filtering out or reporting duplicate lines in sorted input. It can be customized with various options to display line counts, show only duplicates or unique lines, and ignore case or specific fields/characters. By mastering these options, you can efficiently handle text data in Linux files and streams.
