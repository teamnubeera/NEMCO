Linux - Text File Operations

Chapter 3: Text File Operations

Overview of Commands:

  • cat: Concatenate and display file contents.
  • more: View file contents interactively (forward).
  • less: View file contents interactively (forward/backward).
  • head: Display the first few lines of a file.
  • tail: Display the last few lines of a file.
  • cut: Remove sections from each line of files.
  • paste: Merge lines of files.
  • grep: Search for patterns in files (regular expressions).
  • sed: Stream editor for filtering and transforming text.
  • awk: Text processing tool for pattern scanning and processing.
  • sort: Sort lines of text files.
  • uniq: Report or omit repeated lines.
  • wc: Word, line, and byte count for files.
  • diff: Compare files line by line.
  • cmp: Compare two files byte by byte.
