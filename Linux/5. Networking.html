Linux - Networking

Networking Commands

Overview of Commands:

  • ping: Send ICMP echo request to a network host.
  • ifconfig: Display or configure network interfaces (older tool).
  • ip: Show/manipulate routing, devices, and tunnels.
  • netstat: Print network connections, routing tables, etc.
  • ss: Display socket statistics (replacement for netstat).
  • traceroute: Trace the route packets take to a network host.
  • nslookup: Query DNS servers for information.
  • dig: Query DNS information in more detail.
  • wget: Non-interactive download utility (HTTP, FTP, etc.).
  • curl: Transfer data from or to a server (supports multiple protocols).
  • scp: Secure copy files between hosts over SSH.
  • rsync: Remote file and directory synchronization.
  • ftp: File transfer protocol client.
  • ssh: Secure shell to login to remote machines.
  • ip addr: Show IP address information of interfaces.
