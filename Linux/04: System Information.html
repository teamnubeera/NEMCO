Linux - System Information

Chapter 4: System Information

Overview of Commands:

  • uname: Print system information (kernel, machine, etc.).
  • hostname: Display or set the system's hostname.
  • uptime: Show how long the system has been running.
  • top: Display active processes and system performance.
  • htop: Interactive process viewer (enhanced version of top).
  • ps: Report a snapshot of current processes.
  • df: Report file system disk space usage.
  • du: Estimate file space usage.
  • free: Display memory usage (RAM and swap).
  • vmstat: Report virtual memory statistics.
  • dmesg: Print system messages, especially from kernel ring buffer.
  • lscpu: Display information about the CPU architecture.
  • lsblk: List information about block devices (e.g., disks).
  • iostat: Report CPU and input/output statistics.
  • lspci: List PCI devices.
  • lsusb: List USB devices.
