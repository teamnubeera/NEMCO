who: Show Who is Logged into the System

The who command in Linux is used to display information about users who are currently logged into the system. It provides details such as the username, terminal, login time, and the originating IP address or hostname for each user session.

Basic who Syntax

The basic syntax of the who command is:

who [options]

Where:

  • [options] are optional flags to modify the behavior of the command.

Examples

Show All Logged-in Users

To display a list of all users currently logged into the system, simply run:

who

Example output:

john     pts/0        2025-01-17 08:30 (:0)
alice    pts/1        2025-01-17 09:00 (:1)

This will show the username, terminal, login time, and originating IP/hostname (if applicable).

Show Detailed Information with -a Option

The -a option displays additional information about each user, including the idle time, process ID, and more:

who -a

Example output:

john     pts/0        2025-01-17 08:30 (:0)  00:10  1234  (192.168.1.100)
alice    pts/1        2025-01-17 09:00 (:1)  00:05  5678  (192.168.1.101)

Show Logged-in Users for a Specific Terminal

To show who is logged into a specific terminal (e.g., pts/0), use the following syntax:

who pts/0

Example output:

john     pts/0        2025-01-17 08:30 (:0)

Show Information for a Specific User

To see information about a specific user (e.g., john), you can specify the username:

who john

Example output:

john     pts/0        2025-01-17 08:30 (:0)

Show the Current System Time with -b Option

The -b option displays the system's last boot time:

who -b

Example output:

         system boot  2025-01-16 17:30

Useful who Options

  • -a: Show all available information, including idle time, process ID, and more.
  • -b: Display the system's last boot time.
  • -d: Show only users with a current, active session.
  • -q: Display only the usernames of logged-in users, one per line.
  • -u: Show the idle time of each user.
  • -h: Display help information for the who command.

Security Considerations

While the who command provides important information about logged-in users, be cautious when sharing this data, especially on multi-user systems. Sensitive data could be exposed, such as which users are currently logged in and where they are logged in from.

Note: The who command shows publicly available information about logged-in users and does not require elevated privileges (root access) to run.

Conclusion

The who command is a useful tool for checking who is logged into a Linux system and viewing various details about their sessions. Whether you need to monitor user activity or troubleshoot, who is an essential command for system administrators and power users alike.

For more advanced usage and additional options, refer to the official who manual page.
