w: Show Who is Logged in and Their Activities

The w command in Linux displays information about users who are currently logged in and their activities. This includes details such as the username, the terminal they are logged into, their login time, idle time, and the processes they are running. This command is particularly useful for system administrators to monitor user activity on the system.

Basic w Syntax

The basic syntax of the w command is:

w [options] [username]

Where:

  • [options] are optional flags that modify the behavior of the command.
  • [username] is the name of a specific user to show information for (optional).

Examples

Show Information About All Logged-in Users

To display a list of all users currently logged into the system along with their activities, simply run:

w

Example output:

 12:34:56 up 10 days,  4:25,  3 users,  load average: 0.15, 0.20, 0.30
USER     TTY      FROM             LOGIN@   IDLE   JCPU   PCPU  WHAT
john     pts/0    :0               08:30    1.00s  0.05s  0.00s  w
alice    pts/1    192.168.1.100    09:15    3.00s  0.10s  0.01s  bash
bob      pts/2    192.168.1.101    10:00    5.00s  0.15s  0.05s  vim

This output provides details such as:

  • USER: The username of the logged-in user.
  • TTY: The terminal (e.g., pts/0) the user is using.
  • FROM: The IP address or hostname of the user's machine.
  • LOGIN@: The time at which the user logged in.
  • IDLE: The amount of time the user has been idle.
  • JCPU: The total time used by all processes attached to the terminal.
  • PCPU: The time used by the current process.
  • WHAT: The command or process the user is currently running.

Show Information for a Specific User

To display information for a specific user (e.g., john), use the following command:

w john

Example output:

 12:34:56 up 10 days,  4:25,  3 users,  load average: 0.15, 0.20, 0.30
USER     TTY      FROM             LOGIN@   IDLE   JCPU   PCPU  WHAT
john     pts/0    :0               08:30    1.00s  0.05s  0.00s  w

Show Information with Additional Options

The w command also provides various options to customize the output.

Show System Uptime

The -u option displays system uptime (this is often shown by default):

w -u

Show Output for a Specific Terminal

To show information for a specific terminal, use the terminal identifier (e.g., pts/0):

w pts/0

Show Information in Short Format

The -h option hides the header row, showing only the information about users:

w -h

Useful w Options

  • -h: Omit the header from the output.
  • -u: Show system uptime (this is often displayed by default).
  • -s: Show short output (less detailed than the default).
  • -f: Show the full login information (including the user's hostname).
  • -i: Display idle time in minutes instead of seconds.
  • -V: Show version information for the w command.
  • -h: Display help information about the w command.

Security Considerations

While the w command provides useful information about logged-in users and their activities, it can reveal sensitive details about system usage. Ensure that access to this information is restricted to trusted users, especially on multi-user systems.

Note: The w command does not require root access to run, but users with elevated privileges can see more detailed information, such as which users are running specific processes.

Conclusion

The w command is an invaluable tool for administrators and users to track who is logged in and what activities they are performing on a Linux system. Whether you want to monitor user behavior or troubleshoot issues, w provides a detailed, real-time overview of user activity.

For more advanced usage and additional options, refer to the official w manual page.
