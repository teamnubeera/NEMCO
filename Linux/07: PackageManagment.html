Linux Package Management Tools

Debian-Based Distributions

  • apt-get: APT package management tool (Debian-based distros).
  • apt-cache: Search for packages (Debian-based distros).
  • dpkg: Debian package manager (Debian-based distros).

RPM-Based Distributions

  • yum: Package manager (RPM-based distros like CentOS).
  • dnf: Next-generation package manager (Fedora, RHEL 8+).
  • rpm: RPM package manager (for RPM-based distros).

Arch Linux

  • pacman: Package manager for Arch Linux.
